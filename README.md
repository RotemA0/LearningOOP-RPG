# Learning OOP RPG

A text-based RPG game built in C# while learning Object-Oriented Programming.

## About
This project demonstrates core OOP concepts including classes, inheritance, polymorphism, and encapsulation through a playable turn-based combat game.

## Features Implemented
- ✅ Character base class with inheritance
- ✅ Player and Enemy derived classes  
- ✅ Turn-based combat system
- ✅ Attack, defense, and health management
- ✅ Victory rewards (experience and gold)

## Planned Features
- 🚧 Items and inventory system
- 🚧 Dungeon exploration
- 🚧 Leveling system
- 🚧 Boss battles
- 🚧 Berserk-themed Dragon Slayer side quest

## What I Learned
- Creating and using classes
- Inheritance and base classes
- Method overriding with `virtual` and `override`
- Object-oriented design patterns
- Combat loop logic

## Tech Stack
- C# .NET
- Console Application

## How to Run
1. Open solution in Visual Studio
2. Press F5 to run
3. Follow on-screen prompts to battle enemies
